Api BookCamp
========================

Prérequis
------------

* PHP 8.1.0 ou plus

Installation
------------

```cmd
git clone https://github.com/charlottesaidi/NFSBookCamp/api.git
cd bookcamp_api/
composer install
```

Utilisation
-----

**1** Lancer le serveur symfony avec cette commande :

```cmd
symfony serve
```

Accéder à l'URL donnée (<http://localhost:8000> par défaut).
